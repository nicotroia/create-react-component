# React Component Scaffolder

## Installation 
  - `npm install -g https://github.com/nicotroia/create-react-component.git`

## Usage
  - `create-react-component init`

## Test
 - `node index.js init`

## Input
 - name (Sidebar)
 - dir (src/components/)
 - hasStyles (Y/n)
 - hasActions (Y/n)
 - hasConstants (Y/n)
 - hasReducers (Y/n)
 - hasSelectors (Y/n)

## Output
  - src
    - components
      - Sidebar
        - index.js
        - Sidebar.js
        - sidebar.styl
        - actions.js
        - constants.js
        - reducer.js
        - selectors.js
